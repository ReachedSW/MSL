# MSL
Marty(Sama) Standard Library for C++17

It includes:
- <msl/check_assert.h>
	- msl::check_assert
- <msl/file_ptr.h>
	- msl::file_ptr
- <msl/inherit_cast.h>
	- msl::inherit_cast<T>
- <msl/truncate_cast.h>
	- msl::truncate_cast<T>
	- msl::truncate<T>

### How to use it
```cpp
#include <msl/msl.h>
```
